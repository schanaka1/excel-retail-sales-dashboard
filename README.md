# Retail Sales Performance Dashboard (Excel)

## Project Overview
This project analyzes retail sales data using Advanced Microsoft Excel to create an interactive sales dashboard.

## Tools Used
- Microsoft Excel
- Power Query
- Power Pivot
- DAX
- Pivot Tables
- Slicers

## Dataset
100+ retail sales records including region, product, category, and revenue details.

## Key Features
- Automated data cleaning using Power Query
- Data modeling with Power Pivot
- Interactive dashboard with slicers
- KPI metrics for revenue, orders, and units sold
